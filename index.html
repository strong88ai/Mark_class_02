Part 1 — 50분+

Google Flow로 실전 이미지 제작

한 번에 4장, 무제한 무료, 하나의 프롬프트로 어디서든 활용할 수 있습니다.

Core Concept
Google Flow = 이미지 생성 + 편집 + 영상까지 하나로 통합된 AI 창작 스튜디오
한번 프롬프트를 입력하면 4장이 동시에 생성됩니다. 이미지 생성은 크레딧을 소모하지 않아 무제한 무료입니다.

참고: labs.google/flow (Google Flow 공식)

예시 미술 도구함을 생각해보세요. 예전에는 물감 가게, 붓 가게, 도화지 가게를 따로 돌아다녀야 했는데,
이제 Flow 하나에 물감(이미지 생성), 붓(편집), 도화지(영상)가 다 들어있는 겁니다.

프롬프트 구조 5요소

AI에게 이미지를 요청하는 글(프롬프트)에 이 5가지를 구체적으로 적을수록 원하는 결과에 가까워집니다.

분위기전체적인 감정/톤따뜻하고 아늑한 느낌

1부에서 만든 이미지에 QR코드 아트를 결합하면 스마트 안내 시스템이 완성됩니다.

Core Concept
QR코드 아트 = 예쁜 그림이 들어있지만 스캔 기능은 100% 동일한 QR코드
종이 안내문에 QR코드를 하나 붙이면 스캔 시 영상, 지도, 전화 연결까지 가능합니다.
종이 안내문이 스마트 안내문으로 업그레이드되는 거예요.

참고: qrcode-ai.com / me-qr.com

예시 현관문을 생각해보세요. 예쁜 스테인드글라스 문이어도 열고 닫는 기능은 똑같죠? QR코드 아트도 마찬가지예요.

실습: 1부 이미지로 QR코드 아트 만들기

1부에서 만든 이미지를 QR코드 안에 넣어서 나만의 QR코드 아트를 만듭니다.

1

1부에서 만든 이미지 준비

1부 실습에서 다운로드한 브랜드 캐릭터, 안내 이미지 등을 준비합니다

2

AI QR코드 아트 생성

Hugging Face QR Code AI Art Generator 접속 (무료) → QR코드에 연결할 URL 입력 + 1부 이미지 업로드 → 생성 → 스마트폰으로 스캔 테스트

3

AI로 생성한 이미지, 이렇게 활용하세요
